# SmartCalc – Interactive Financial Learning Platform

SmartCalc is a modern, interactive financial calculator platform designed for the **FinCal Innovation Hackathon**. It moves away from traditional spreadsheet-style tools to provide a visually engaging and educational experience for everyday investors.

## 🚀 Key Features

- **Educational Focus**: Built to teach users about compounding, inflation, and systematic investing.
- **5 Compliant Modules**:
  - **SIP Calculator**: Visualize long-term wealth creation.
  - **SWP Calculator**: Plan systematic withdrawals with corpus depletion mapping.
  - **Top-Up SIP**: See how annual step-ups accelerate financial goals.
  - **Goal-Based Investing**: Inflation-adjusted planning for life's milestones.
  - **Retirement Planner**: Multi-phase accumulation and withdrawal visualization.
- **Modern UI**: Sleek glassmorphism design with brand-compliant colors (`#224c87`, `#da3832`).
- **Real-time Visualization**: Dynamic Area, Pie, and Bar charts powered by `recharts`.
- **Accessibility**: Built to **WCAG 2.1 AA** standards with full ARIA support and semantic HTML.
- **Client-Side Priority**: Fast, secure calculations performed entirely in the browser.

## 🛠️ Tech Stack

- **Framework**: Next.js 15 (App Router)
- **Styling**: Vanilla CSS (Custom Variable System)
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Charts**: Recharts

## 📖 How to Run

1. **Clone the repository**:
   ```bash
   git clone https://github.com/iamshresthraj/SmartCalc.git
   cd SmartCalc
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Run the development server**:
   ```bash
   npm run dev
   ```

4. **Open the app**:
   Navigate to [http://localhost:3000](http://localhost:3000)
